Privacy Policy for BoboBot

Effective Date: June 12, 2026

This Privacy Policy describes how BoboBot ("we", "our", or "the Bot") collects, uses, and handles information when you use the Bot in a Discord server.

1. Information We Collect

A. Information Stored Locally on Server

To provide core functionalities, the Bot stores the following data in a local database (SQLite) on our hosting server:

  • Custom Commands: Custom trigger keywords and their corresponding response text/media URLs defined by server administrators or users.
  • User Long-Term Memory: Personalized facts or preferences that you consent to save (managed via the !記憶 / /記憶 commands). This is processed using the Mem0 framework and stored locally.
  • Server Configuration: Settings such as enabled features and prefix settings.

B. Information Processed in Real-Time (In-Memory)

The Bot processes the following data temporarily in RAM to respond to events. This data is not stored permanently:

  • Message Content: Read in real-time to match command prefixes, detect typos (for spelling correction roasts), find specific link patterns (for auto-embed fixes), and detect repeated messages (for echo/copypasta features).
  • Channel History: Up to 50 recent messages are read when you chat with the Bot (!bobo or @mentions) to provide conversational context to the Google Gemini API.

2. Third-Party Data Sharing

  • AI Conversation: Message text and images sent to the AI chat function are transmitted to the Google Gemini API to generate replies. We do not use this data to train machine learning models.
  • Image/Link Parsing: URLs are processed through public APIs (such as Yahoo Finance API for stock inquiries, Saucenao for image search, or Pixiv/Wnacg/NHentai for preview retrieval). Only the relevant parameter (e.g., stock ticker, URL) is sent.

3. Data Control and Deletion

Users have absolute control over their data:

  • Disabling Memory: You can turn off memory tracking at any time by typing !記憶 關閉 or /記憶 關閉.
  • Deleting Memory: You can clear all your stored memory by typing !記憶 清除 or /記憶 清除.
  • Custom Commands: Server administrators can remove any custom command using the !remove [command] instruction.

4. Contact

If you have any questions or data deletion requests, please contact the bot owner (焦糖波波) through the Discord server.
